使用 Github Pages 和 Hexo 搭建个人博客

Created2020-11-02|Updated2021-01-27
|Post View:

Hexo搭建教程

搭建博客分类

1.可以在CSDN，博客园，简书等平台上直接发表博客。

2.搭建个人博客网站，需自己购买域名和服务器，也需要自己定期维护。

3.将我们的博客部署到github page上。

Hexo简介

Hexo 是一个基于Node.js的静态博客框架。（GitHub官方建议使用Jekyll博客生成工具）使用Markdown（或其他标记语言）编写帖子，Hexo会生成带有精美主题的静态文件。

Hexo搭建步骤

  1. 安装Git
  2. 安装Node.js
  3. 安装Hexo
  4. 将博客部署到 Github Pages 上

1.安装Git

可前往官网下载安装相应版本

![image-20210127141326465](D:\program files\YEZ_Blog\blog\source\images\image-20210127141326465.png)

如果下载慢，也可下载淘宝git镜像

![image-20210127141545927](D:\program files\YEZ_Blog\blog\source\images\image-20210127141545927.png)

配置Git

1.在Git上设置你的邮箱和用户名。（打开Git目录的git-cmd.exe）

2.在GitHub账户中添加你的公钥。（打开Git目录的git-bash.exe）

在Github的settings中找到SSH and GPG keys设置选项，点击New SSH key
将你的id.rsa.pub里面的信息复制进去（Title任意）

![image-20210127170449908](D:\program files\YEZ_Blog\blog\source\images\image-20210127170449908.png)

2.安装Node.js

可前往官网选择LTS下载安装相应版本

![image-20210127134815696](D:\program files\YEZ_Blog\blog\source\images\image-20210127134815696.png)

如果下载慢，也可下载淘宝node.js镜像

![image-20210127140521602](D:\program files\YEZ_Blog\blog\source\images\image-20210127140521602.png)

3.安装Hexo

  • 创建blog文件夹，右键选中文件夹，点击Git Bash Here
  • 创建hexo文件夹，右键选中文件夹，点击Git Bash Here

初始化完成后会生成下列文件

![image-20210127144558973](D:\program files\YEZ_Blog\blog\source\images\image-20210127144558973.png)

  • 启动本地博客服务（不要关闭窗口）

可以使用http://localhost:4000/访问建好的博客网站

4.将博客部署到 Github Pages 上

  • 登入Github，创建一个新的仓库（要求仓库名为YourGithubName.github.io）

![image-20210127152237403](D:\program files\YEZ_Blog\blog\source\images\image-20210127152237403.png)

  • 安装将hexo部署到Github page的git插件deployer
  • 修改hexo文件夹下的配置文件_config.yml

修改#Deployment中部分内容
